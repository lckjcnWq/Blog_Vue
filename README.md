# VUE博客管理系统

## 项目简介

通过vue+elementUi建立的一个博客系统,提供给三方用户学习vue框架知识点  

开发环境:webStorm

## 主要实现的功能

1. 分类管理
2. 评论管理
3. 标签管理
4. 有情管理
5. 编辑博客
6. 密码管理

## 前端

- Vue.js2.5
- Vue-Router3.0
- axios0.19.0

## 默认

用户名和密码默认 admin
